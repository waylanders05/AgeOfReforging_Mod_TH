# AoR Thai Mod

มอดภาษาไทยสำหรับเกม **Age of Reforging: The Freelands**

## ดาวน์โหลด

ดาวน์โหลดไฟล์มอดพร้อมเล่น:

[AoRThaiMod_v1.0.0.zip](dist/AoRThaiMod_v1.0.0.zip)

## วิธีติดตั้ง

1. แตกไฟล์ `AoRThaiMod_v1.0.0.zip`
2. เปิด `AoRThaiInstaller.exe`
3. กด `ติดตั้ง / อัปเดตมอด`
4. เปิดเกม แล้วไปที่ `Options` -> `Language` -> เลือก `ไทย`

ตัวติดตั้งจะพยายามหาโฟลเดอร์เกมจาก Steam ให้อัตโนมัติ ถ้าหาไม่เจอให้กดเลือกโฟลเดอร์เกมเอง

## วิธีถอนมอด

เปิด `AoRThaiInstaller.exe` แล้วกด `ถอนมอดออก`

## หมายเหตุ

- มอดนี้ไม่แก้ไขไฟล์เกมต้นฉบับ
- อัปเดตเกมผ่าน Steam ได้ตามปกติ
- ถ้าเกมมีแพตช์ใหม่ ข้อความใหม่บางส่วนอาจกลับเป็นภาษาอังกฤษชั่วคราวจนกว่าจะมีมอดเวอร์ชันใหม่
- เป็นมอดแฟนเมด ไม่เกี่ยวข้องกับผู้พัฒนา/ผู้จัดจำหน่ายเกม
